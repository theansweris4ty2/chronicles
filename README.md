# <div align="center"> *<u>CHRONICLES*</u></div>
<hr>

<p> A turn based straregy game where you build and defend an empire. Each round you get points to spend on three actions. You choose actions from 6 basic actions and 5 advanced actions. Player receives points to spend on those actions based on three dice rolls. The points from each die must be used on separate actions. If you choose the same action in consequtive turns the costs of that action increases by 1 point from the cost of the action in the preceding round. Certain actions can only be taken in tiles that match the type of action chosen.</p>

<p> Every turn there is a random chance a tile will be invade if the tile is unoccupied it will immediately be concurred and the invaders will spawn from that time every round and every round the number spawned will increase until the tile is recaptured. </p>
