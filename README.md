## *CHRONICLES*
> A turn based straregy game where you build and defend an empire. You choose from 6 actions each round and spend points on those actions from dice rolls. If you choose the same action in consequtive turns the costs of that action increases by 1 point from the cost of the action in the preceding round. 

> Every turn there is a random chance a tile will be invade if the tile is unoccupied it will immediately be concurred and the invaders will spawn from that time every round and every round the number spawned will increase until the tile is recaptured.
